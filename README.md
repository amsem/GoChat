# GoChat

## Run Locally

Clone the project

```bash
  git clone https://github.com/amsem/GoChat
```

Go to the project directory

```bash
  cd real-time-chat-app
```

### Back End Setting

Install dependencies

```bash
  cd backend
  go mod tidy
```

Start the server

```bash
  go run .
```

### Front End Setting

Install dependencies

```bash
  cd frontend
  npm install
```

Start the server

```bash
  npm start
```
